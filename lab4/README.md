# Лабораторна робота №4: Інтерактивна фільтрація сигналу

## Опис
Ця програма моделює гармонічний сигнал із накладеним на нього шумом та дозволяє інтерактивно застосовувати цифровий фільтр Баттерворта для очищення сигналу.

## Технології
- **Python**
- **NumPy** — генерація сигналу та шуму.
- **SciPy** — реалізація низькочастотного фільтра Баттерворта (`butter`, `filtfilt`).
- **Matplotlib** — побудова графіків та створення інтерактивного GUI (повзунки, кнопки).

## Як користуватися
Запустіть скрипт `lab4.py`. У вікні, що з'явиться, можна:
- Змінювати амплітуду, частоту та фазу базового сигналу.
- Налаштовувати параметри шуму (зміщення та дисперсію).
- Керувати частотою зрізу (Cutoff Freq) фільтра Баттерворта.
- Вмикати/вимикати відображення зашумленого сигналу.
