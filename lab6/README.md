# ЗПАД Лабораторна робота №6 - OpenCV Обробка Відео

Програма для захоплення відео з веб-камери та його обробки в реальному часі за допомогою бібліотеки OpenCV. Проєкт має модульну архітектуру (C++) та налаштований для автоматичної збірки через CMake.

## Вимоги до системи
* **ОС:** Linux (тестувалось на Ubuntu 22.04 / 24.04 LTS)
* **Залізо:** Наявність підключеної веб-камери (або налаштоване прокидання USB у віртуальну машину)
* **ПЗ:** CMake, G++, Make, OpenCV 4+

## Інструкція зі збірки та запуску

Проєкт містить зручні bash-скрипти для автоматизації розгортання. Для запуску на чистій системі виконайте наступні кроки в терміналі:

1. Надайте права на виконання скриптам: 

chmod +x preinstall.sh build.sh run.sh

Встановіть необхідні залежності та бібліотеки (потребує прав sudo):
./preinstall.sh

Скомпілюйте проєкт:
./build.sh

Запустіть програму:
./run.sh

Керування та інтерактивні елементи
Програма підтримує зміну режимів обробки на льоту за допомогою клавіатури, а також взаємодію за допомогою миші.

Клавіатура (Режими обробки):
1 - Звичайний режим (Normal)

2 - Інверсія кольорів (Invert)

3 - Виділення контурів (Canny Edge Detection)

4 - Розмиття (Gaussian Blur)

5 - Дзеркальне відображення по горизонталі (Mirror)

6 - Ефект симетрії (Center Mirror - дві правих частини кадру)

ESC - Вихід з програми

Миша:
Ліва кнопка миші (LMB) - Намалювати червоний маркер (коло) у місці кліку.

Права кнопка миші (RMB) - Очистити екран від усіх маркерів.

Інтерфейс:
Trackbar (Повзунок) - Дозволяє плавно регулювати яскравість відеопотоку. Значення за замовчуванням — 50.
