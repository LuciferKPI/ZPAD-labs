# Лабораторна робота №5: Створення веб-додатку за допомогою Streamlit

## Опис
Цей веб-додаток створений для інтерактивного аналізу та візуалізації вегетаційних індексів (VCI, TCI, VHI). Програма дозволяє фільтрувати дані за роками, тижнями та областями, а також зручно порівнювати показники різних регіонів.

## Технології
- **Python**
- **Streamlit** — для створення інтерактивного веб-інтерфейсу.
- **Pandas** — для обробки та фільтрації даних.
- **Matplotlib** — для побудови графіків динаміки та порівняння.

## Як запустити
1. Встановіть необхідні залежності: `pip install -r requirements.txt`
2. Запустіть додаток командою: `streamlit run lab5.py`
3. Перейдіть за посиланням, яке з'явиться у терміналі (зазвичай `http://localhost:8501`).
