## Огляд проекту
Ця робота присвячена аналізу та візуалізації реального датасету про затримки авіарейсів. Метою було не просто побудувати графіки, а навчитися очищувати дані та знаходити приховані залежності між різними параметрами польотів.

## Обраний датасет
**Dataset:** [Flight Delays Data](https://raw.githubusercontent.com/PacktPublishing/Pandas-Cookbook/master/data/flights.csv)  
Це багатовимірний датасет, що містить інформацію про рейси США, включаючи:
- **Категоріальні дані:** Авіакомпанії (AIRLINE), аеропорти (ORG_AIR, DEST_AIR).
- **Числові дані:** Дистанція (DIST), затримки вильоту/прибуття (DEP_DELAY, ARR_DELAY), час у повітрі (AIR_TIME).

## Технологічний стек
- **Python 3.12**
- **Pandas** — для маніпуляцій з даними та очищення.
- **NumPy** — для математичних операцій.
- **Matplotlib** & **Seaborn** — для створення статичних та багатовимірних графіків.

## Виконані кроки
1. **Data Cleaning:** Видалення некоректних значень (NaN) та підготовка числових типів даних.
2. **Univariate Analysis:** Побудова гістограми розподілу рейсів за дистанцією (5 діапазонів).
3. **Bivariate Analysis:** Аналіз залежності часу польоту від відстані (Scatter plot).
4. **Multivariate Visualization:** - Теплова карта (Heatmap) кореляції числових ознак.
   - Розподіл затримок за авіакомпаніями (Boxplot).
   - Динаміка середніх затримок по днях тижня та днях місяця (Barplot/Lineplot).
